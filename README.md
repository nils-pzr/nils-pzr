# About

Hi, I'm Nils — a Junior Software Developer from Germany.

I build modern web applications, CMS-based websites and clean digital products with a strong focus on practical value, polished interfaces and maintainable implementation. My current work combines frontend development, Plone / Volto CMS, React, Next.js, TypeScript, UI/UX design and AI-assisted development workflows.

At Interaktiv GmbH, I work on real software projects and gain practical experience across frontend development, CMS development, AI-powered applications, agile workflows and technical collaboration.
